# Books (Category: Books)
A SwiftUI application for NYT books.

## Technologies Used
- Language: Swift 5.*
- Tool: XCode 14.2
- Data: JSON
- Design Patterns: Observable, State, Codable, MVVM, Async/Await, etc.
- Database: Realm
- Tests: XCUnitTest
